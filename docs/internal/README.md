# Internal docs

Planning, product, and UX documents that inform the direction of Sery
Link but are not user-facing reference material.

These are kept in the repo so contributors can read the reasoning
behind design decisions, but they are **work-in-progress** and may go
out of date between releases. If a doc here conflicts with the code,
trust the code.

For user-facing documentation, see the main `README.md` and the public
docs at [sery.ai/docs](https://sery.ai/docs).
